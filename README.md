# SAE 105 – Traitement des données  
## Étude de l’évolution de la part du nucléaire dans la production d’électricité en France

### Contexte
Ce projet s’inscrit dans le cadre de la SAE 105 – Traitement des données.  
Il s’appuie sur les donnéesde la RTE (Réseau de Transport d’Électricité) concernant la production d’électricité en France.

Lien vers la plateforme de données :
https://analysesetdonnees.rte-france.com/

### Problématique
L’objectif de ce projet est d’étudier l’évolution de la part du nucléaire dans la production d’électricité française au fil des années, et d’analyser sa place par rapport aux autres modes de production.

### Données utilisées
Les données proviennent des fichiers CSV publiés par la RTE, notamment :
- Production d’électricité par filière
- Données agrégées à l’échelle nationale
- Données couvrant plusieurs années

Ici, on détaillera surtout la problématique du nucléaire et les fichier qui porte dessus.

### Méthodologie
Le traitement des données est réalisé en Python à l’aide des bibliothèques :
- `math` pour la manipulation des données
- `matplotlib`  pour la visualisation

Les analyses envisagées incluent :
- Calculs de moyennes et de pourcentages
- Étude de l’évolution temporelle
- Comparaison entre le nucléaire et les autres filières de production
- Visualisation sous forme de graphiques (courbes, histogrammes)
